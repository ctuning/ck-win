This repository contains minimal CK components for Windows:
* os:windows*
* os:mingw*

Note that CK component "os:windows-32" contains GPL-licensed tools 
to ensure the compatibility of CK workflows with Linux:
* 7z.exe
* adb.exe
* bzip2.exe
* curl.exe
* diff.exe
* gzip.exe
* md5sum.exe
* ninja.exe
* patch.exe
* sha1sum.exe
* tar.exe
* unzip.exe
* wget.exe
* zip.exe
